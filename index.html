Kubernetes Api Server 未授权访问漏洞
2018-12-05

0x00 漏洞原因

Kubernetes 的服务在正常启动后会开启两个端口：

  1. Localhost Port （默认8080）
  2. Secure Port （默认6443）

这两个端口都是提供 Api Server 服务的，一个可以直接通过 Web 访问，另一个可以通过 kubectl 客户端进行调用。
